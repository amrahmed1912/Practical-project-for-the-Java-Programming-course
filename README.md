# Student GPA Calculator
## Project Description
<p>
  This project is a GPA Calculator with a graphical user interface (GUI) using Java Swing.<br>
  Students can enter their name, ID, and total CreditHours.<br>
  Add multiple subjects with credit hours and grades.<br>
  Calculate the final GPA and display it directly in the interface.<br>
</p>

## Example Usage
<p>
  <b> Enter Student Name: <b>Amr Ahmed <br>
  <b>Enter Student ID:</b> 12345 <br>
  <b>Enter Total Hours: </b> 18 <br>
  <b>Add Subjects: </b> <br>
      - Subject Name: Programming 2 <br>
      - Credit Hours: 4 <br>
      - Grade: 95 <br>
Click Add Subject for each course. <br>
Click Calculate GPA to display the final GPA. <br>
</p>
